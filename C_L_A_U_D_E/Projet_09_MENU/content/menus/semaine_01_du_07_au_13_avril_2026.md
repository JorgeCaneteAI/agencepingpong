## BATCH COOKING — DIMANCHE MATIN

| Heure | Action | Équipement | Résultat |
|-------|--------|------------|----------|
| 9h00 | Lentilles vertes 300g + oignon + carotte + cumin | Cocotte minute (15 min) | Boîte 1 : lundi déj + mercredi déj |
| 9h05 | Riz complet 300g | Casserole (20 min) | Boîte 2 : mardi souper + jeudi déj + vendredi déj |
| 9h05 | Œufs durs ×5 | Casserole (10 min) | Bocal : toute la semaine |
| 9h20 | Cuisses de poulet herbes de Provence | Air fryer 190° (25 min) | Boîte 4 : vendredi déj + samedi déj + samedi souper |
| 9h25 | Soupe de carottes 500g + oignon + curry + bouillon | Cocotte minute (10 min) | Boîte 3 : mercredi souper + jeudi souper |
| 9h30 | Asperges vapeur 8 min + épinards 3 min | Vapeur / micro-ondes | Frais : semaine |
| 9h45 | Mixer soupe de carottes | Blender | — |
| 10h00 | Mise en boîtes + rangement frigo | — | — |

### Récap frigo
- **Boîte 1** : Lentilles → lundi déjeuner + mercredi déjeuner
- **Boîte 2** : Riz complet → mardi souper + jeudi déjeuner + vendredi déjeuner
- **Boîte 3** : Soupe de carottes → mercredi souper + jeudi souper
- **Boîte 4** : Poulet → vendredi déjeuner + samedi déjeuner + samedi souper
- **Bocal** : Œufs durs → toute la semaine
